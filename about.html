Le projet

Un outil fait pour
simplifier le quotidien.

FreshList est né d'un besoin simple : avoir une liste de courses claire, organisée par catégorie, avec les bonnes quantités — sans les fonctionnalités superflues qu'on n'utilise jamais.

Construit en HTML, CSS et JavaScript pur, sans dépendances, il démontre qu'une application web utile et agréable peut être développée avec les fondamentaux du web.

Nos principes

Ce qui guide le projet

01

Simplicité

Aucune fonctionnalité inutile. Chaque élément de l'interface existe pour une raison précise.

02

Clarté

L'information s'affiche de façon lisible, sans surcharge visuelle ni distractions.

03

Performance

Zéro dépendance externe. Le site charge instantanément, fonctionne hors ligne après le premier chargement.

Stack technique

Construit avec les fondamentaux

Ce projet est volontairement construit sans framework ni bibliothèque. HTML sémantique, CSS personnalisé avec des variables et JavaScript vanilla — une démonstration des compétences fondamentales du développement web.
